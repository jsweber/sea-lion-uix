# `react-blockquote`

Blockquote is a component for displaying quoted content, with multiple sizes and styles.

## Installation

```sh
pnpm add @sea-lion/react-blockquote
# or
npm install @sea-lion/react-blockquote
yarn add @sea-lion/react-blockquote
```

## Usage

Import the component in your code:

```tsx
import { Blockquote } from '@sea-lion/react-blockquote';
```

### Basic Usage

```jsx
<Blockquote>Quoted content</Blockquote>
<Blockquote size="3" color="blue">Quote with size and color</Blockquote>
```

Supports sizes `"1"` through `"9"` and multiple color themes.
