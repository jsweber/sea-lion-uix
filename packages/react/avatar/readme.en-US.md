# `react-avatar`

Avatar is a component for displaying user or entity avatars, with multiple sizes and styles.

## Installation

```sh
pnpm add @sea-lion/react-avatar
# or
npm install @sea-lion/react-avatar
yarn add @sea-lion/react-avatar
```

## Usage

Import the component in your code:

```tsx
import { Avatar } from '@sea-lion/react-avatar';
```

### Basic Usage

```jsx
<Avatar src="https://example.com/avatar.jpg" alt="User avatar" />
<Avatar fallback="AB" />
```

### Sizes and Styles

Supports sizes `"1"` through `"9"`, variants `solid` | `soft`, radius `none` | `small` | `medium` | `large` | `full`, and multiple color themes.
