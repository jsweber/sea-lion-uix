# `react-badge`

Badge is a component for displaying labels, counts, or status as short text.

## Installation

```sh
pnpm add @sea-lion/react-badge
# or
npm install @sea-lion/react-badge
yarn add @sea-lion/react-badge
```

## Usage

Import the component in your code:

```tsx
import { Badge } from '@sea-lion/react-badge';
```

### Basic Usage

```jsx
<Badge>Label</Badge>
<Badge variant="solid" color="red">Important</Badge>
<Badge size="2">Count</Badge>
```

### Sizes and Variants

Supports sizes `"1"` | `"2"` | `"3"`, variants `solid` | `soft` | `surface` | `outline`.
