# `react-code`

Code is a component for displaying code snippets, with multiple styles and size options, making code stand out from surrounding content.

## Installation

```sh
pnpm add @sea-lion/react-code
# or
npm install @sea-lion/react-code
yarn add @sea-lion/react-code
```

## Usage

Import the component in your code:

```tsx
import { Code } from '@sea-lion/react-code';
```

### Basic Usage

```jsx
<Code>const x = 1;</Code>
<Code size="2" variant="soft">Code snippet</Code>
```

## When to Use

- Display code snippets or commands in text
- Highlight technical names or variables
- Mark code syntax in docs or tutorials
- Show config keys, parameter names, or technical terms
