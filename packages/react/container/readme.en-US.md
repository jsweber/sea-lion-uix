# `react-container`

Container is a component that constrains maximum content width and centers content on the page.

## Installation

```sh
pnpm add @sea-lion/react-container
# or
npm install @sea-lion/react-container
yarn add @sea-lion/react-container
```

## Usage

Import the component in your code:

```tsx
import { Container } from '@sea-lion/react-container';
```

### Basic Usage

```jsx
<Container size="3">Main page content area</Container>
```

Supports props such as `size` to control max width.

## When to Use

- Create page layouts with a standard width
- Keep line length readable for long text on large screens
- Center content with consistent margins
